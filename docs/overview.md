- triadic hash-state evolution
- deterministic payload canonicalization
- explicit genesis block
- recomputation-based validation
- coherence metrics
- CLI run/validate flow

Later layers:
- signed transactions
- wallet identities
- mempool
- node networking
- coherence-aware consensus
- contract execution
